# Export And Delivery

La matriz recomienda una separacion clara:

- export basico en `Free`
- export listo para entrega en `Pro`

## Checklist de salida

- [ ] Portada correcta
- [ ] Secciones en orden
- [ ] Diagramas visibles
- [ ] Formulas visibles
- [ ] Imagenes visibles
- [ ] Titulos consistentes

## Tabla de formatos

| Formato | Uso comun |
| --- | --- |
| PDF | compartir version final |
| HTML | publicar en web |
| DOCX | iteracion con editores externos |
| EPUB | lectura larga en dispositivos |

> [!IMPORTANT]
> Este arbol existe para que la exportacion por carpeta sea demostrable desde samples, no solo desde documentacion.
