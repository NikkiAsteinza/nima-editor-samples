# Core Workflow

Esta seccion cubre la base del producto: escribir, revisar y sincronizar documentacion tecnica sobre archivos reales.

## Incluye

- edicion de contenido
- revision visual y tecnica
- flujo Git sobre la rama actual
- historial basico por archivo

> [!NOTE]
> La matriz trata este bloque como el minimo util para que la app sea viable como herramienta de documentacion.
