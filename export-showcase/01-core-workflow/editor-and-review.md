# Editor And Review

Nima Editor esta pensado para trabajar directamente sobre Markdown y carpetas normales.

## Validaciones utiles en esta pagina

1. cambio entre modo enriquecido y texto plano
2. preservacion de headings y listas
3. lectura del contenido antes de exportar

## Tabla de revision

| Paso | Que revisar |
| --- | --- |
| Write | contenido correcto |
| Review | preview y estructura |
| Diff | cambios exactos |
| Restore | recuperacion basica |

> [!WARNING]
> Antes de exportar, conviene revisar que tablas, callouts y bloques tecnicos mantengan su forma final.
